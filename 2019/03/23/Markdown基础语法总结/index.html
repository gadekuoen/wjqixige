Markdown基础语法总结

发表于 2019-03-23 分类于 Markdown语法基础

一、概述

Markdown就是一个把文本转换称html格式的一个工具，网上有好多类似的博客，看起来眼花缭乱，我在阅读别人博客的同时自己总结一份，以便之后查看。

二、常用语法

1、标题

####1.1 上下文标题:
一级标题
=======

二级标题
-------

####1.2 [h1~h6]:
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2、强调

1. *斜体* 或 _斜体_
2. **粗体**
3. ***加粗斜体***

3、代码

1. 代码块标记
    ```
2. 代码块缩进表示法
    tab 或 四个空格
3. 语法高亮显示
    ```javascript
4. 内联代码块
    `

4、表格

####4.1 写法一：
| a | b | c |
|:—–:|:——|——:|
| 居中 | 左对齐| 右对齐|
|=======|=======|=======|
####4.2 写法二：
直接上html代码好了，推荐

<table border="1">
    <tr>
        <td>row 1, cell 1</td>
        <td>row 1, cell 2</td>
    </tr>
    <tr>
        <td>row 2, cell 1</td>
        <td>row 2, cell 2</td>
    </tr>
</table>

5、链接

####5.1 内链式
百度1{:target=”_blank”}
其中： [] 链接文字
() 链接地址
{} 跳转方式（注：兼容性一般）

####5.2 引用式
[百度2][2]{:target=”_blank”}
[2]: http://www.baidu.com/ “百度2下”
####5.3 邮箱链接
xxxx@163.com

6、图片

6.1 内联式

![图片alt](图片地址 "title")

6.2 引用式

![图片alt][标记]
[标记]:地址链接 "title"

6.3 带链接

内联式：
    [![图片alt(可省略)](图片地址 "title")](链接地址){跳转方式}
引用式：
    [![图片alt(可省略)](图片地址 "title")[标记]{跳转方式}]
    [标记]: 链接地址

####1.1 无序

2、清单

- [x] 选项一
- [ ] 选项二

3、引用

在被引用的文本前加上 > 符号

这是引用1
这是引用2

这是引用3

4、脚注

脚注名字[^注脚名字] 
注：注脚与注脚之间需要空一行

markdown[^1]

[^1]：markdown是一种纯文本标记语言
